# 1 远程登录

## 1.1 服务器地址

| 通道 | IP 地址       | 端口 | 协议 |
| ---- | ------------- | ---- | ---- |
| 内网 | 192.168.1.123 | 2000 | SSH  |
| 公网 | 暂无          | 暂无 | 暂无 |



## 1.2 SSH登录

​	在 Linux / Windows 本地终端使用以下命令，其中 user 为登录用户名：

```shell
ssh user@192.168.1.123 -p 2000
```

​	服务器返回输入密码提示：

``` 
user@192.168.1.123's password:
```

​	输入密码后，出现命令提示符，即表示登录成功：

``` shell
user@ILISA:~$
```



## 1.3 修改密码

 使用以下命令修改密码：

```shell
passwd
```
